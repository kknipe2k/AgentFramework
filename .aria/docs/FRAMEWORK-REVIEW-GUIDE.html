Skill Importance & Grading Criteria

Why Skills Matter

Skills define HOW Claude should behave. Incomplete skills lead to inconsistent behavior, missed HITL checkpoints, and poor output quality.

Professional Standard

Professional skills should include: clear triggers, explicit outputs, mode variations, error handling, examples, and integration points.

Underdeveloped Skills Risk

researcher.md, slide-generation.md, and context-refresh.md are too light and may lead to unpredictable AI behavior.

Critical Code Issues

Bash 4+ Associative Arrays
ralph.sh:41-42
declare -A story_failures

Windows Git Bash ships with bash 3.x. Associative arrays require bash 4+.

Fix: Use file-based storage or simple indexed arrays.

Error Swallowing Pattern
ralph.sh:483
output=$(echo "$full_prompt" | claude --dangerously-skip-permissions -p $model_flag 2>&1 | tee /dev/stderr) || true

The || true swallows ALL errors. No way to detect Claude crash vs success.

Fix: Use set -o pipefail and proper error handling.

Token Estimation Inaccurate
model-selector.sh:485-486
input_tokens=$(( ${#full_prompt} / 4 ))
output_tokens=$(( ${#output} / 4 ))

Rough estimate. Claude tokenizer averages ~3.5 chars/token but varies by content.

Fix: Use tiktoken library or Claude's tokenizer API.

URL Injection Risk
verify-executor.sh:29
APP_URL="${APP_URL:-http://localhost:3000}"
# Later used in:
curl -s -o /dev/null -w "%{http_code}" "$APP_URL"

If APP_URL contains shell metacharacters, could inject commands.

Fix: Validate URL format before use.
