New Features (v2.0-2.1 - Jan 2026)

Test Suite (v2.1)

20 tests: 15 unit, 3 integration, 2 validation. Cross-platform Python runner with Git Bash path conversion.

Meta-Reasoning

Systematic decision-making with Thompson Sampling for model selection. Uses learned priors from past sessions.

Offline RL

Learning pipeline that extracts episodes from signals/decisions, calculates rewards, and updates Beta priors.

Deep Research

Web research with 5 HITL gates: depth, strategy, queries, checkpoint, synthesis. Includes confidence scoring.

Signal Schema v2

8 signal types (tool, skill, agent, decision, verification, error, HITL, session) with rich context.

Skill Importance & Grading Criteria

Why Skills Matter

Skills define HOW Claude should behave. Incomplete skills lead to inconsistent behavior, missed HITL checkpoints, and poor output quality.

Professional Standard

Professional skills should include: clear triggers, explicit outputs, mode variations, error handling, examples, and integration points.

Underdeveloped Skills Risk

researcher.md, slide-generation.md, and context-refresh.md are too light and may lead to unpredictable AI behavior.

✓ FIXED: URL Injection Risk

# Validates format: https?://[a-zA-Z0-9]
# Blocks dangerous chars: ;\|&$\`><(){}[]!#

OPEN: File Locking for State Files
common.sh:499+ (emit_signal, emit_decision)

State file operations use atomic append but NO flock-based file locking. Works for serial execution but NOT safe for concurrent tool invocations.
