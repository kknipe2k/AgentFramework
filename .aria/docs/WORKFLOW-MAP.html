All Skills (12 Total)

Core Skills (All Modes)

planning ALL

Break work into tasks with estimates

Triggers: "plan this", start of work
Output: current-plan.json

executing ALL

Implement tasks with verification gates

Triggers: Approved plan exists
Output: Code, commits

debugging ALL

Fix test failures and errors

Triggers: Test failure, error
Output: Working code

discovery ALL

Explore existing codebase

Triggers: Modify flow, unfamiliar codebase
Output: project-context.md

tdd ALL

Test-driven development

Triggers: "write tests first", "tdd"
Output: Tests → Implementation

Extended Skills (STANDARD+)

brainstorming STANDARD+

Explore options before planning

Triggers: "brainstorm", "explore options"
Output: IDEA.md

prototyping FULL+

Visual/structural mockups

Triggers: "mockup", "prototype"
Output: .aria/prototypes/*.html

tracking STANDARD+

Monitor progress and metrics

Triggers: During/after execution
Output: progress.json, metrics

context-refresh STANDARD+

Reset after drift/failures

Triggers: 3+ failures, between phases
Output: Handoff summary

report-writer STANDARD+

End-of-workflow summary

Triggers: All tasks complete, "summary"
Output: Summary + dashboard offer

Mode Comparison

Feature LITE STANDARD FULL FULL+
Task count 1-5 6-15 16-40 40+
Hierarchy Tasks only Phases → Tasks Steps → Phases → Tasks Epics → Steps → Phases → Tasks
Planning Quick (optional) Yes Yes + risks Yes + design doc
Brainstorming Optional Yes Yes + research Yes + prototyping
Verification If tests exist Every task Mandatory Mandatory
HITL checkpoints Destructive only Risky actions All risky Per epic + risky
Design notes No Key decisions All reasoning All + architecture
Failure threshold 1 → stop 2 → stop 3 → escalation 3 → escalation
Context refresh No Between phases Between major steps Between epics
End-of-workflow report Minimal Yes + dashboard offer Full + dashboard Full + architecture validation
