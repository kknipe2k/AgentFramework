## Key Insights

1. **Memory is the Moat** — Every project requires sophisticated memory management, from sliding context windows to knowledge graphs with decay
2. **The Agentic Loop is the Core Primitive** — Plan → Execute → Test → Reflect appears in all 5 projects at increasing scale
3. **HITL is Non-Negotiable** — No project suggests fully autonomous operation; knowing when to involve humans is the mark of production
4. **Explainability as Architecture** — If your agent can't explain its reasoning, it's not production-ready
5. **Resource Awareness** — Production AI has a cost per inference; ignoring it is how startups die
