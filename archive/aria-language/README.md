# ARIA Language (Archived)

This folder contains archived documentation and examples for the **ARIA Programming Language** concept - a hybrid language design combining Python and TypeScript features.

This is **separate from** the ARIA Orchestration System (located in `/.aria/`) which is an autonomous AI development framework.

## Contents

### Documentation
- `ARIA_LANGUAGE_SPEC.md` - Full language specification
- `ARIA_SYNTAX.md` - Syntax reference
- `ARIA_ORCHESTRATION.md` - Language orchestration concepts
- `ARIA_RAILS.md` - Language safety rails
- `ARIA_CLAUDE_INTEGRATION.md` - Claude AI integration design
- `ARIA_COMPLETE.md` - Complete language overview
- `IMPLEMENTATION_ROADMAP.md` - Implementation plan
- `CLAUDE.md.template` - Template for Claude context

### Examples
- `examples/01_basics.aria` - Basic syntax examples
- `examples/02_async.aria` - Async/await patterns
- `examples/03_components.aria` - Component system
- `examples/04_classes.aria` - Class definitions
- `examples/05_interop.aria` - Interoperability

## Status

This language design work is archived. The active project in this repository is the **ARIA Orchestration System** - see `/.aria/docs/` for current documentation.
