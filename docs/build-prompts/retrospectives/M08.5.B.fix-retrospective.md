- [ ] Candidate gotcha (§15) — **synthetic-JS DnD bypasses the Tauri OS handler.** A test that drives HTML5 drag-and-drop via `browser.execute()` with a synthetic `DataTransfer` event sequence will PASS on `main` (with `dragDropEnabled: true`) because the JS-dispatched events never reach the OS-level handler — they go directly to the DOM event loop. To regression-guard a Tauri-shell drag-handler defect, the test mechanism MUST drive real OS pointer events (W3C WebDriver Actions API multi-step pattern). The author MUST cite the external upstream reference inline + name the rationale, so the next stage can cite this stage's file as the in-project upstream. A `gotcha #32` corollary.
