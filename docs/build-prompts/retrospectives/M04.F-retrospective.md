- [ ] **`fireEvent.click` on `type="submit"` doesn't propagate to `onSubmit` in jsdom.** Pattern: "When testing a form-submit flow in vitest, `fireEvent.click(saveButton)` where the button is `type='submit'` inside a form with `onSubmit` does NOT trigger the form submit handler in jsdom ‚Äî the click handler runs but the submit cycle is skipped.
