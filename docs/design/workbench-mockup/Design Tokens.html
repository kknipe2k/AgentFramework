Five node kinds — AA proof on a light canvas

Rule 4: distinct node kinds are visually distinct by color and weight. Each kind owns a hue used three ways — a 4px accent bar, a glyph chip, and a label color verified AA on white. This is where light themes go muddy; the contrast ratios below prove the five stay legible and separable.

Two-axis model. Kind is one axis (above). Runtime status is a separate axis — shown by the node ring/glow + a status dot — so an active Tool node reads as “Tool” (teal bar) and “running” (blue pulse) without the two signals colliding.
